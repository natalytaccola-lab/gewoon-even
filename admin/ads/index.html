Kill & Scale Rules

Automatic decisions

🔴 Kill If

  • Spent >€20 with 0 conversions
  • CTR <0.5% after 2,000+ impressions
  • CPC >€4.50 after 500+ clicks (3× target)
  • Running 7+ days with consistently declining trend

🟢 Scale If

  • CTR >2% AND CPC <€1.50 for 3+ days
  • Method: duplicate winning creative into other ad sets FIRST
  • Then bump budget +20% max per day
  • Never scale a creative running <3 days

Troubleshooting — Before You Panic

Common issues

😕 "No clicks after 24h"

Normal. Meta's learning phase takes ~48h. If still 0 by hour 48, pause the ad set and check: is the ad actually approved? (Ads Manager → Delivery column)

💸 "CPM is sky high (€40+)"

Audience too narrow OR creative fatigued. Try broadening interests. If CPM >€50 week 1 → likely a delivery issue, not creative.

📉 "CTR good, CPC bad"

Hook works but landing page leaks. Check quiz start → completion rate. If <50%, quiz is the problem, not the ad. Check upsell page conversion if quiz is fine.

🔥 "One ad crushing, others dying"

Normal — it's why we test. Kill the losers, duplicate the winner into the other ad sets with small copy tweaks. Don't edit the winner — it resets learning.

🎯 "Clicks but no conversions"

Clicks come cheap; conversions need trust. Check that quiz loads fast on mobile, upsell page loads under 3 seconds, and that Stripe checkout (iDEAL) is not broken (test it yourself from mobile with a real Dutch bank account).

⏱️ "Should I let it run overnight?"

Yes — especially for this product. Late evening to early morning delivery (20:00 CET – 02:00 CET) is exactly when Dutch anxiety/insomnia sufferers are scrolling and susceptible. Don't day-part to daytime only.
